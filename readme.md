# Projekt LaTeX z Docker i VS Code

![Docker](https://img.shields.io/badge/Docker-Desktop-2496ED?logo=docker&logoColor=white)
![VS Code](https://img.shields.io/badge/VS%20Code-Visual%20Studio%20Code-007ACC?logo=visualstudiocode&logoColor=white)
![LaTeX](https://img.shields.io/badge/LaTeX-TeX-008080?logo=latex&logoColor=white)

Ten projekt służy do kompilacji dokumentów LaTeX w środowisku VS Code z wykorzystaniem kontenera Docker.

## Wymagania

- **Docker Desktop** – zainstaluj z [docker.com](https://www.docker.com/products/docker-desktop) i upewnij się, że jest uruchomiony.
- **Visual Studio Code** – zainstaluj z [code.visualstudio.com](https://code.visualstudio.com).
- **Rozszerzenie LaTeX Workshop** – zainstaluj w VS Code (Extensions) wyszukując „LaTeX Workshop” (autor: James Yu).

## Konfiguracja

1. **Uruchom Docker Desktop** i poczekaj, aż będzie działał.
2. **Zainstaluj LaTeX Workshop** w VS Code (Ctrl+Shift+X → wyszukaj „LaTeX Workshop” → Install).
3. **Otwórz projekt** w VS Code.

## Użycie

- Otwórz projekt w VS Code.
- Kompilacja odbywa się w kontenerze Docker.

## Sposób uruchomienia

1. Uruchom Docker Desktop.
2. W katalogu projektu uruchom plik `start_w_Dockert.bat`.
3. Po starcie kontenera pracuj w VS Code i kompiluj dokument.

## Rozwiązywanie problemów

- Jeśli kompilacja nie działa, sprawdź czy Docker Desktop jest uruchomiony.
